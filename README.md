## Installation
```shell
npm install @babel/parser @babel/types babel-eslint eslint eslint-config-ultra-refined --save-dev
```

### Setup
In `package.json` file:
```json
{
  "eslintConfig": {
    "extends": "ultra-refined"
  }
}
```
Or in `.eslintrc*` file:
```json
{
  "extends": "ultra-refined"
}
```
